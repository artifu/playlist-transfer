FAQ

Questions before you move a playlist.

Short answers for the current public-link MVP.

Does PlaylistXfer require a Spotify login?

No for the current MVP. PlaylistXfer is built around public Spotify playlist links, so you can paste a share link without connecting a Spotify account.

Can it transfer private Spotify playlists?

Not in the public-link MVP. Private playlist support would require Spotify authorization and a different product flow.

When do I connect Apple Music?

At the create step. You can preview and match first, then authorize Apple Music only when you are ready to write the destination playlist.

Does it add songs to my library or download them?

It creates a playlist from Apple Music catalog tracks. It does not download audio files, and it is not intended to upload local music.

Why are some tracks marked review or missing?

Apple Music can have different versions of the same track. PlaylistXfer keeps fuzzy results out of the final playlist unless you approve them.

How many tracks can I transfer?

The web MVP analyzes the full playlist by default, up to the current product limit shown in the transfer scope control. Larger playlists can take longer because matching runs in batches.

Is PlaylistXfer affiliated with Spotify or Apple Music?

No. PlaylistXfer is independent and uses those names only to describe the source and destination services.

What if the public Spotify link fails?

Try a normal public playlist you own or can edit, then paste that link. Spotify public surfaces can change, so the app includes a fallback guide when a link cannot be read.
