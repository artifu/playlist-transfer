Feedback makes the transfer better.

Use this page for support, bugs, sponsor interest, and product feedback.

Bug reports

If a playlist fails to preview, match, or create, include the public Spotify playlist URL, the step that failed, and whether you were using web or iOS. Open an issue in the PlaylistXfer GitHub issues.

Product feedback

Useful feedback includes confusing labels, bad match examples, slow playlists, Apple Music authorization issues, and places where the product felt stuck.

Sponsor or affiliate interest

The web page reserves room for a future sponsor or affiliate placement, but the transfer path stays first. Sponsor messages should be optional, disclosed, and away from Apple Music authorization.

Privacy questions

Read the Privacy Policy for the current MVP data handling notes. The app is designed to avoid storing Apple Music user tokens long term.
